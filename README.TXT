TT4BP Recent SiteWide Posts Widget creates a recent sitewide posts widget for buddypress that includes the ability for you to filter out posts from specific blogs by passing the blog IDs of the blogs to exclude

== Description ==

TT4BP Recent SiteWide Posts Widget creates the same widget as the buddypress recent sitewide posts widget only this widget includes the ability for you to exclude posts from spefified blog IDs. 

For more information and support please visit http://themestown.com

== Installation ==

Backup your WordPress database and files before you install or update. 

1. 	Upload the folder tt4bp-recent-sitewide-posts with its contents to wp-content/plugins/

2.   	Activate the plugin

3.	Go to "Appearance" then "Widgets". Drag the "TT Recent Sitewide Posts Buddypress" widget to the desired position

4. 	Enter the ID or IDs of the blog or blogs that you don't want any posts from to show up in the recent sitewide posts

5. 	Save your settings.


If you are having problems please visit http://www.themestown.com for help.

Hint: If you prefer to hide the images that show up in the excerpts, add class  .activity-content img { display:none;}  to your CSS file
